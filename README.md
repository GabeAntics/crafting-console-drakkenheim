# Crafting Console - Drakkenheim

A FoundryVTT Module to add Monsters of Drakkenheim recipes to the Crafting Console module. Requires Monsters of Drakkenheim module.

## What This Does

Bring the danger and wonder of Drakkenheim straight into your crafting workflow.

This setup is built to help GMs and players turn monster victories into meaningful crafting progression:
- Parse harvestable components from creature biography content.
- Convert extracted parts into Crafting Console friendly ingredient items.
- Keep rarity, source monster, and provenance attached to every generated ingredient.
- Accelerate recipe creation and campaign bookkeeping for Monsters of Drakkenheim content.

## Macro Spotlight: Harvest Ingredients From Biography

The Harvest Ingredients macro is designed to feel fast, visual, and rewarding at the table:
- Select a token, run the macro, and it resolves embedded biography content.
- Detects Harvestable Components and pre-fills your component choices.
- Uses rarity from the source text, with CR fallback when needed.
- Generates polished ingredient items directly in actor inventory.
- Includes provenance metadata so every ingredient has a clear story.

In short: slay the monster, run the macro, and watch your crafting inventory come alive.

## Requirements

- Foundry VTT
- Crafting Console module
- Monsters of Drakkenheim module

## Suggested Workflow

1. Open a creature that includes harvestable component information.
2. Select its token in the scene.
3. Run Harvest Ingredients From Biography.
4. Confirm component selections and rarity.
5. Create items and use them in Crafting Console recipes.
